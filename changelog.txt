- PDF 편집: 미리보기 영역을 아래 손잡이로 끌어 크게/작게 (세로 크기 조절)
- PDF 편집: Ctrl+휠로 페이지(썸네일) 크게/작게 — 한 줄에 보이는 장수 단위로 또렷하게 조절, 휠 줌 버벅임(렉) 제거
- PDF 폴더 일괄통합(쪽수 매기기): 회전된(가로 스캔 등) 페이지의 쪽 번호가 옆/위에 찍히던 문제 수정 — 항상 페이지 하단 중앙에 표기

- 'PDF 편집' 도구 추가 — 페이지를 썸네일로 보며 직접 편집
  · 페이지 삭제·회전, 다른 PDF를 원하는 위치(맨 뒤 / N쪽 뒤)에 끼워넣기
  · 드래그로 페이지 순서 바꾸기(무손실), 더블클릭으로 크게 미리보기
  · 편집 결과는 새 PDF로 저장(원본 보존), 끼워넣은 페이지는 색으로 구분 표시
- 시작 화면(스플래시) 추가 — 기동 단계 표시
- 실행 파일 경량화(약 88MB)로 시작 속도 개선

[6.4]
- 폴더 일괄병합: 2단계 이상 병합 시 저장 위치에도 원본 폴더 구조를 그대로 재현 (이전엔 결과가 한곳에 평면 저장)
- 변환 진행 표시 개선: 한글처럼 오래 걸리는 변환도 '변환 중… (N초 경과)'로 살아있음 표시 (멈춤/작업중 구분)

[6.3]
- 화면 전면 개편: 좌측 메뉴 + 홈 화면(도구 카드·최근 작업), 새 앱 아이콘
- HWP → HWPX 변환 도구 추가 (원본 .hwp는 휴지통 이동 옵션, 기본 켬)
- HWP → PDF의 Microsoft Print to PDF 방식이 저장 창에서 멈추던 문제 해결, 변환 속도 대폭 개선
- 모든 도구에서 파일/폴더 드래그 앤 드롭 지원 (끌어오면 창이 앞으로)
- 수백 개 파일 목록도 빠르게 표시, 행 아무 곳이나 클릭해 선택
- 글자 선명도 개선 (전용 글꼴 내장)
- 오류 발생 시 진단 로그 전송 기능 (동의 시에만)
- 자잘한 개선: 유형 체크박스 가독성, 옵션 배치 정리, 업데이트 안내 간소화, PDF 분할·병합 아이콘 교체
